Installation und Deinstallation

Überblick

Macam besteht aus zwei Teilen: Ein unabh&aumml;ngiges Programm und eine QuickTime-Treiber-Komponente. Beide können unabhängig voneinander installiert und benutzt werden. Wenn Sie jedoch beide Teile verwenden, stellen Sie sicher, daß beide Teile die gleiche Version haben.

Installeren des Programmes

Die Benutzung des Programmes benötigt keine explizite Installation. Doppelklicken Sie das Programmicon, um es zu starten (wenn Sie dies lesen, haben Sie dies wahrscheinlich schon getan). Trotzdem ist es eine gute Idee, das Programm an einen Ort zu kopieren, an dem Sie auch Ihre anderen Programme aufbewahren (beispielsweise in den "Programme"-Ordner).

Deinstallieren des Programmes

Um das Programm zu entfernen, löschen Sie es einfach (wie eine normale Datei).

Installieren der QuickTime-Komponente

Um andere, QuickTime-kompatible Webcam-Programme mit ihrer Kamera zu benutzen, müssen Sie zuvor die QuickTime-Komponente installieren. Ziehen Sie dazu das "macam.component"-Icon in den Ordner "/Library/QuickTime/" (der Ordner "QuickTime" im Ordner "Library" auf der Hauptebene der Festplatte, von der Sie gestartet haben - wenn noch kein "QuickTime"-Ordner existiert, erstellen Sie einen neuen Ordner und benennen Sie ihn exakt so um). Wenn Sie eine ältere Version von "macam.component" installiert hatten, entfernen Sie diese zuvor (siehe weiter unten).

Deinstallieren der QuickTime-Komponente

Zur Deinstallation der QuickTime-Komponente befolgen Sie folgende Schritte: Bewegen Sie "macam.component" von "/Library/QuickTime/" in den Papierkorb. Starten Sie Ihren Rechner neu. Löschen Sie anschließend den Papierkorb.

Komplette Deinstallation

Die Deinstallation des Programmes und der Komponente ist normalerweise vollkommen ausreichend. Es können lediglich einige Voreinstellungsdateien auf Ihrem Rechner verbleiben. Um auch diese zu entfernen, öffnen Sie Ihren Home-Ordner ("Privat"). Öffnen Sie darin den Ordner "Library" und darin "Preferences". Löschen Sie innerhalb dieses Ordners alle Dateien, die mit "net.sourceforge.webcam-osx" beginnen.

Alternativen

Wenn Sie die macam-Komponente keinen anderen Benutzern zur Verfügung stellen wollen oder wenn Sie nicht über die notwendigen Administrator-Zugriffsrechte verfügen, können Sie anstatt der "/Library/QuickTime/" die Library in Ihrem "Privat"-Ordner verwenden. Die restlichen Installations- und Deinstallationsschritte bleiben gleich.
